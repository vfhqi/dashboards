Notes for review

  • Compact (default): 38px wide × 36px tall. Pill only. Date + URL surface on hover. ~10 rows visible per screen at desktop. iPad portrait fits ~14 columns visible at once before horizontal scroll kicks in.
  • Stuck rings pulse subtly to draw the eye (look at row 3 — DCC has 3 stuck cells).
  • Audit border colour is independent of state — a published memo can fail audit (HTRO row, Q14 has green-✓ inside red border).
  • Cancelled cells (HTRO row, Q10) show grey strikethrough — Richard cancelled the SS-reports research on 28-Apr.
  • Hover behaviour: popover appears with full detail. Real implementation would close on mouse-out / tap-elsewhere.
  • Empty cells are non-clickable, dimmed, indicate "not requested for this stock."
  • Re-runs show in expanded view as version count + link to archived versions (HTRO Q14 v2 example).
